# 🧠 Quiz Interativo para Programadores

Um projeto web interativo desenvolvido com **HTML, CSS e JavaScript básico**, com o objetivo de testar e reforçar conhecimentos em programação e desenvolvimento web.

---

## 📌 Sobre o Projeto

Este projeto consiste em um **quiz online** com perguntas variadas voltadas para iniciantes e estudantes da área de tecnologia.

O foco principal é praticar:

* Estruturação com HTML
* Estilização com CSS
* Uso de formulários
* Interações básicas com JavaScript

---

## 🚀 Funcionalidades

O sistema de quiz inclui diferentes tipos de entrada de dados:

* ✅ Múltipla escolha (radio buttons)
* ✍️ Campo de texto
* 🔒 Campo de senha
* 📅 Seleção de data
* ☑️ Checkboxes
* 📂 Upload de arquivo
* 🔽 Menu suspenso (select)
* 🖼️ Identificação por imagem

### Recursos adicionais:

* 📊 Tabela de pontuação com feedback
* 📋 Respostas ocultas com `<details>`
* 💬 Formulário de feedback com alerta
* 🎨 Interface estilizada com CSS
* 📱 Layout adaptável (responsividade básica)
* 🔗 Página extra "Sobre mim"

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta:

```bash
cd quiz
```

3. Abra no navegador:

```
index.html
```
