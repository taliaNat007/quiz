# 🧠 Quiz Interativo para Programadores

Um site simples e interativo desenvolvido em HTML para testar conhecimentos básicos em programação e desenvolvimento web.

## 📌 Sobre o Projeto

Este projeto consiste em um **quiz online** com diferentes tipos de perguntas voltadas para programadores iniciantes e intermediários.

O objetivo é praticar conceitos de HTML enquanto o usuário testa seus conhecimentos sobre:

* Linguagens de programação
* Conceitos básicos da web
* Estrutura de páginas
* Boas práticas

---

## 🚀 Funcionalidades

O quiz possui diversos tipos de entrada de dados:

* ✅ Múltipla escolha (radio buttons)
* ✍️ Campo de texto
* 🔒 Campo de senha
* 📅 Seleção de data
* ☑️ Checkboxes
* 📂 Upload de arquivo
* 🔽 Menu suspenso (select)
* 🖼️ Identificação por imagem

Além disso:

* 📊 Tabela de pontuação com feedback
* 📋 Lista com respostas corretas
* 📱 Layout adaptável (meta viewport)
